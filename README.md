# soma-recursiva
# exercicio desenvolvido para à aula de estrutura de dados do Professor Leandro Colevati

Criar uma aplicação em Java que tenha uma função recursiva que calcule o somatório dos N primeiros número NATURAIS (a função deve retornar zero para números negativos).

• O Código deve apresentar, em formato de comentário, como foi definida a condição de parada;

• O Código deve apresentar, em formato de comentário, como foi definida a relação de chamada dos passos;
